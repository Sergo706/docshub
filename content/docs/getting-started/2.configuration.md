# Configuration

Learn how to configure the overarching settings and environment variables for the DocsHub project.

## Environment Variables

Copy the `.env.example` file to `.env` and fill in the required values.

```bash [Terminal]
cp .env.example .env
```

| Variable | Description | Default |
| --- | --- | --- |
| `NUXT_PUBLIC_SITE_URL` | The public URL of your application. | `http://localhost:3000` |
| `DATABASE_URL` | The connection string for your Postgres database. | `""` |
| `SESSION_SECRET` | A secure, random string used to sign session cookies. | `""` |

## App Config

You can override UI settings and application branding inside the `app.config.ts` file located in the root of the project.

```typescript [app.config.ts]
export default defineAppConfig({
  appName: 'DocsHub',
  ui: {
    primary: 'green',
    gray: 'neutral',
  }
})
```

> [!IMPORTANT]
> Some database operations will fail if the `DATABASE_URL` is not set or points to an invalid endpoint.
